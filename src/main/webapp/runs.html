Перевод страниц на китайский

Requirement references
Требования: Китайский язык
Требования: Поиск
Description

Тест проверяет перевод текстов на странице на китайский язык : все страницы.

Preconditions

Пользователь залогинен в системе.

Steps
Step Expected Result
Логинимся в сервис под пользователем admin/amdin Пользователь залогинен в системе: в header отображается логотип пользователя
Кликаем на меню изменения языка и выбираем китайский язык Все тексты на странице переведены на китайский язык
Переходим на страницу поиска элементов На странице присутствуют блоки: реклама, элементы объявлений. Все тексты в блоках переведены на китайский. Ни в одном месте не встречается английский
